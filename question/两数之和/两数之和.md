## 暴力搜索
双重for循环
时间复杂度为O(n*n)
### 使用哈希表
首相创建一个map，键为nums[i]，值为[i]
对于遍历到的元素先检查target-v是否在哈希表中，如果在就说明找到了两数之和等于target的两个元素，进行append
```go
func twoSum(nums []int, target int) []int {
	var hash =map[int]int{}
	for i,v:=range nums{
		
		if p,ok:=hash[target-v];ok{
			return []int{i,p}
		}
		hash[v]=i
	}
	return nil

}
```
